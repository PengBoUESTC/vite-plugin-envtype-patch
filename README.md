# pkg-template

## init

```bash
yarn
```

## build

```bash
yarn build
```

## release

1. rewirte release config file `release.config.ts`, set your `releaseUser` & `scripts`
2. run command
```bash
yarn release
```

## note

1. init jest config file 
```bash
npx jest --init
```

2. simple git hooks init 
```bash
npx simple-git-hooks
```

3. init eslint config
```bash
npm init @eslint/config
```
